# Run passive model

# In micronole
1. Run the terminal with root user
   `sudo -s`
   Password : nvidia

2. Run the following launchfile
   `roslaunch qcar passive.launch`

# In Cockpit
1. Run the terminal with root user
   `sudo -s`
   Password : nvidia

2. Run the following launchfile
   `roslaunch passive passive.launch`
